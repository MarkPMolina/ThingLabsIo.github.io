# ESP8266 Weather Station
In this workshop you will begin by learning the basics of working with an ESP8266 dev board and LUA. You will start by writing the IoT version of 'Hello, World' Next, you will build a _Thing_ that reads temperature and humidity. Once you have the _Thing_ working you will connect it to an Azure IoT Hub. Finally you will create an Azure Stream Analytics job that will process the live data stream and output it to a website for reporting with Javascript charts. 

To complete this workshop you will need an Azure ESP8266 kit. You will also need a Microsoft Azure account (free trial is fine).

Estimated time to complete: 15-minutes.

# Disney World Chat Bot
In this workshop you will build a simple bot using Node.js, the Bot Framework, and a LUIS model. Once the bot is complete you will be able to test the bot out using the Bot Framework Channel Emulator. Finally you will learn about managing session data so that your bot can remember context between questions.

Estimated time to complete: 15-hours.
